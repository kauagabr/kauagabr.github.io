Projeto 1

O projeto consiste em implementar a Cifra de César, um método de criptografia clássico, em duas partes distintas.

Parte 1 - Cifragem:
A primeira parte envolve a criação de um algoritmo capaz de cifrar uma mensagem de texto. O usuário fornece um documento contendo a mensagem e escolhe um deslocamento, que determina quantas posições cada letra da mensagem original será deslocada no alfabeto. O programa então aplica a Cifra de César à mensagem, gerando um novo documento com a mensagem cifrada.

Parte 2 - Descriptografia sem a Chave:
Na segunda parte, o projeto explora a tentativa de descriptografar uma mensagem cifrada sem conhecer o deslocamento original. Primeiro, o programa calcula a frequência das letras na mensagem cifrada para identificar as letras mais comuns. Em seguida, determina os possíveis deslocamentos com base nessas letras mais comuns. Utilizando os possíveis deslocamentos, o programa tenta descriptografar a mensagem cifrada e exibe os resultados possíveis. No geral, o projeto abrange a aplicação da Cifra de César para cifrar mensagens, seguida por uma abordagem de análise de frequência para tentar descriptografar a mensagem cifrada sem a chave de deslocamento original. É uma exploração prática dos conceitos de criptografia clássica e análise de frequência, oferecendo insights sobre a segurança relativa desse método de criptografia.

Tecnologias utilizadas: Linguagem de programação Python

Projeto 2

O projeto é um programa de gerenciamento de vendas para um mercado, desenvolvido em Python. Ele permite aos usuários cadastrar produtos, realizar vendas, calcular o valor total das compras, escolher a forma de pagamento e gerar um cupom fiscal. O programa utiliza formatação de cores para melhorar a apresentação no terminal e inclui funções para imprimir informações relevantes, como o cabeçalho do cupom, lista de produtos, detalhes da compra e informações fiscais. Embora o código possa ser otimizado e organizado de maneira mais eficiente, o projeto em si visa automatizar tarefas relacionadas a vendas e fornecer uma experiência mais agradável para os clientes.

Projeto 3

O projeto apresentado é um sistema de gerenciamento para um Pet Shop. O programa é projetado para oferecer uma interface interativa por linha de comando, permitindo que os usuários realizem várias operações relacionadas às atividades do Pet Shop, como agendamento de consultas para diferentes espécies de animais, registro de informações dos clientes e funcionários, além da exibição detalhada dessas informações.

O sistema oferece três principais funcionalidades:

1. Agendamento de Consultas: Os usuários podem agendar consultas para diferentes espécies de animais. O programa coleta informações sobre o animal, como nome, idade e descrição, bem como dados do cliente, incluindo nome, CPF, telefone e endereço. As consultas são associadas aos animais e seus respectivos proprietários.

2. Registro de Funcionários: Os usuários podem adicionar informações de funcionários ao sistema, como nome, idade, CPF, telefone, cargo e salário. Além disso, o sistema registra o endereço do funcionário. Os funcionários adicionados são armazenados em uma lista e podem ser posteriormente exibidos.

3. Exibição de Consultas e Funcionários: O sistema permite a visualização das consultas agendadas, incluindo detalhes sobre os animais e seus proprietários. Além disso, os usuários podem exibir a lista de funcionários cadastrados no Pet Shop, juntamente com seus respectivos cargos. A função de exibição permite ao usuário escolher entre diferentes opções, como exibir todas as consultas ou filtrar por espécie de animal.

O projeto utiliza uma abordagem orientada a objetos, com classes como PetShop, Animal, Cliente, Funcionario e outras para representar os elementos do sistema. As funções de interação com o usuário são implementadas por meio de menus, onde o usuário seleciona opções numéricas para realizar as operações desejadas.

No geral, o projeto visa automatizar a gestão básica de um Pet Shop, permitindo o agendamento e o acompanhamento de consultas, além de manter um registro organizado de funcionários e suas informações. O sistema oferece uma interface simples, mas eficaz, para facilitar a interação dos usuários com as funcionalidades do Pet Shop.

Tecnologias utilizadas: Linguagem de programação Java
